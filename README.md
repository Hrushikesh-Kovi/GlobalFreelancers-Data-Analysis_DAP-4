# 🌍 Global Freelancer Data Analysis

## 📌 Overview
This project performs **Data Cleaning and Exploratory Data Analysis (EDA)** on a global freelancer dataset to understand trends in freelancer demographics, skills, experience, and earnings.

## 🛠 Tools Used
- 🐍 Python
- 🐼 Pandas
- 🔢 NumPy
- 📊 Matplotlib
- 📈 Seaborn
- 📓 Jupyter Notebook

## ⚙️ Work Done
- 🧹 Data cleaning and preprocessing  
- 🔍 Handling missing values  
- 📊 Exploratory Data Analysis (EDA)  
- 📉 Data visualization using Matplotlib and Seaborn  
- 💡 Extracting insights from freelancer data  

## 🔑 Key Insights
- ⚖️ The dataset shows a **balanced gender distribution** among freelancers.  
- 🌎 Freelancers come from **multiple countries**, highlighting the global reach of freelancing.  
- 💰 **Experience has a positive relationship with hourly rate**.  
- ⭐ **Higher ratings often correspond to better client satisfaction**.  
- 💻 Popular freelancer skills include **Web Development, Mobile App Development, and Data Analysis**.
